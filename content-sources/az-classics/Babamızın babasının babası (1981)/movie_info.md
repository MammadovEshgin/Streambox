# Babamızın babasının babası (1981)

## Synopsis
160 yaşlı Əziz baba kəndin fəxridir. Film onun gənclik xatirələri, itmiş məhəbbəti və nəvəsinin xoşbəxtliyi üçün etdiyi fədakarlıqlar ətrafında cərəyan edir.
